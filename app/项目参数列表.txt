客户端sessionStorage、cookieStorage对象键名参数统一参考

sessionStorage{
    'UIF' : 登录成功时暂存的用户信息
    'UID' : 登录成功时暂存的用户ID
    'OID' : 当前系统openID
    'PERMIT' : 权限数据
}

cookieStorage{
    '':''
}

redis{
    'MENU' : 暂存的所有菜单
    'PERMIT' : 权限数据
    'PPER' : 页面权限数据，在controler、model、service能过DataProcess.getUserInfo('PPER',this.uid)读取
    'SYS_OPTIONS' : 系统参数  (于BehaviorModel.loadSysOption() 写入)
}

session{
    "UID" : '请求验证成功时暂存的用户ID'
}

post、get参数参考：
tid : 栏目id
uid : 用户id
oid : 当前系统openID
